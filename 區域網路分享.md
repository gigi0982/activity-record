# 區域網路分享設定

## 立即讓其他人存取的方法

### 1. 查詢您的內網IP
在終端機執行：
```bash
ifconfig | grep "inet " | grep -v 127.0.0.1
```

### 2. 修改後端設定
將伺服器設為監聽所有網路介面：

```javascript
// 在 server/index.js 最後修改
app.listen(port, '0.0.0.0', () => {
  console.log(`伺服器運行於 http://0.0.0.0:${port}`);
});
```

### 3. 分享網址
假設您的內網IP是 192.168.1.100，其他人可訪問：
- `http://192.168.1.100:3000` (網頁版)

### 4. 手機安裝PWA
1. 在手機瀏覽器開啟網址
2. iOS: Safari → 分享 → 加入主畫面
3. Android: Chrome → 選單 → 安裝應用程式

## 注意事項
- ⚠️ 僅限同一WiFi網路下使用
- ⚠️ 重啟路由器後IP可能改變
- ⚠️ 建議設定固定IP位址
