## 快速部署步驟

### 準備工作
1. 註冊 GitHub 帳號
2. 將程式碼推送到 GitHub
3. 選擇雲端服務商

### Vercel 部署 (推薦)
```bash
# 安裝 Vercel CLI
npm i -g vercel

# 在專案根目錄執行
vercel

# 設定環境變數
vercel env add GOOGLE_SHEET_ID
vercel env add FIREBASE_SERVICE_ACCOUNT_KEY
```

### 環境變數設定
- `GOOGLE_SHEET_ID`: 您的 Google Sheets ID  
- `FIREBASE_SERVICE_ACCOUNT_KEY`: Firebase 服務帳號金鑰
- `NODE_ENV`: production
