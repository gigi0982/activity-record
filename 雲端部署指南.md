# 雲端部署快速指南

## 免費雲端部署 - Vercel (推薦)

### 1. 準備工作
```bash
# 安裝 Git (如果尚未安裝)
# 註冊 GitHub 帳號 https://github.com
# 註冊 Vercel 帳號 https://vercel.com
```

### 2. 推送到 GitHub
```bash
cd /Users/dorisyang/Desktop/活動紀錄

# 初始化 Git
git init
git add .
git commit -m "初始提交"

# 連接到 GitHub (需要先在 GitHub 建立倉庫)
git remote add origin https://github.com/你的用戶名/activity-record.git
git push -u origin main
```

### 3. Vercel 部署
1. 登入 Vercel → Import Project
2. 選擇你的 GitHub 倉庫
3. 設定環境變數：
   - `GOOGLE_SHEET_ID`
   - `FIREBASE_SERVICE_ACCOUNT_KEY`
4. 一鍵部署

### 4. 部署後好處
- ✅ 全球任何地方都可存取
- ✅ 自動 HTTPS 安全連線
- ✅ 免費自訂網域 (例：activity-record.vercel.app)
- ✅ 自動備份和版本控制
- ✅ 手機可安裝 PWA

## 替代方案 - Ngrok (臨時分享)

如果想要快速測試，可使用 Ngrok：

```bash
# 安裝 ngrok
brew install ngrok

# 啟動隧道 (在另一個終端)
ngrok http 3001

# 會得到像這樣的公開網址：
# https://abc123.ngrok.io
```

### Ngrok 的限制
- ⚠️ 每次重啟網址會改變
- ⚠️ 免費版有連線時間限制
- ⚠️ 需要保持電腦開機
